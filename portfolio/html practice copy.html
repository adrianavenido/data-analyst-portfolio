This project automated the end-to-end process of generating monthly coaching scorecards for over 180 full-time employees using Google Workspace and Apps Script. This solution replaced the manual, time-intensive workflow with an integrated automation that dynamically generates scorecards, emails them to employees, and ensures governance compliance by storing completed forms in a structured Google Drive folder system.
